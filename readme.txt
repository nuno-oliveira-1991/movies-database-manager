This is an app that you can use to see movie trilers and rate them. If you have access to it you can even extend the movie-trailer database.
It was developed as a react project by Nuno Oliveira during a FullStack Development course at EDIT, Porto.

To run this app:

1 - clone this repository or download it.

2 - install node packages in the root directory (use the command: npm install)

3 - run server app (use the command: npm run dev; or: npm run build)
 
4 - go to the client folder (use the command: cd client)

5 - install node packages (use the command: npm install)

6 - run frontend app (use the command: npm run dev)

7 - Register and have fun exploring the database and rating your favourite movies!!
